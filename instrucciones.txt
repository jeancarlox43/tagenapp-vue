# Tagenapp instaladores e instrucciones


## Instaladores 
[Node.js](https://nodejs.org/en/) 


## Despues de haber instalado node js junto con npm ejecutamos el siguiente comando en el cmd dentro de la carpeta para descargar las dependencias que necesita la aplicación

```
npm install


```

## instalamos la console de vue para ejecutar sus comandos

```
npm install -g @vue/cli

```

### Para ejecutar la aplicación  en un servidor local (localhost) 

```
npm run serve
```

### Para crear una version de produccion de la app
```
npm run build
```
